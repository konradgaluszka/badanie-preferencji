## Provision

Requirements:
- docker
- docker-compose
- yarn (brew install yarn)

in root folder run:
```
$ docker-compose build
$ docker-compose up
```
frontend will be served on
```
http://localhost:3000/
```

backend
```
http://localhost:8000/
```
